# Qodo CLI Installation

## Commands
```bash
npm install -g @qodo/command
qodo login
```

## Requirements
- Node.js and npm must be installed
- Download from: https://nodejs.org/

## Alternative
```bash
# Using yarn
yarn global add @qodo/command
qodo login
```

## Status
- npm not found on PATH
- Install Node.js first
- Then run Qodo commands
